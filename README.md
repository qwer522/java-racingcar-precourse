# 자동차 경주 게임

## 기능 정의
* 사용자의 입력 데이터로 자동차 유효성 검사 (,로 구분 및 5글자 이하), 테스트 케이스 포함 
* 이동 입력 후 랜덤 전진, 테스트 케이스 포함
* 승리자 선정, 테스트 케이스 포함
* 콘솔 출력

## 진행 방법
* 자동차 경주 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.
